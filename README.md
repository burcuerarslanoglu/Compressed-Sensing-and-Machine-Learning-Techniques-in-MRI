# Compressed-Sensing-and-Machine-Learning-Techniques-in-MRI
Advanced Magnetic Resonance Imaging, TU Delft, Q3, Individual Project

# Repository Contents
- main.m: main script
- report.pdf: project report
- MR/: sample MRI images (brain scans)
  
# Summary
Reviewed compressed sensing and machine learning approaches for accelerated MRI, covering techniques and applications across both domains. Implemented a compressed sensing + DnCNN pipeline in MATLAB for brain MRI reconstruction. Showed that ML-assisted undersampling can recover SNR comparable to fully sampled acquisitions while reducing scan time.

# Requirements
- MATLAB R2024b
